## Android studio 工程目录

工程目录在左侧, 切换至 project 后有如下目录

- manifests
  - AndroidManifest.xml : app 内部运行配置

- java 
  - 有三个包, 一个是源码, 其他是测试用的
- res
  - drawable 
  - layout : app 页面布局
  - mipmap : 启动图标
  - values : 常量定义文件
    - strings.xml
    - dimens.xml
    - colors.xml
    - styles.xml
- Gradle Scripts
  - build.gradle  : app 工程编译规则 , 分为项目级和模块级
  - proguard-rules.pro 
  - gradle.properties 
  - settings.gradle
  - local.properties
